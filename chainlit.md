# iPhone User Guide Assistant

A Retrieval-Augmented Generation (RAG) chatbot for the **Apple iPhone User Guide (iOS 7.1)**.

- Answers are grounded **strictly** in the document — no outside knowledge.
- Every answer shows the **page number(s)** and **section** it came from.
- If the answer isn't in the guide, the bot says so instead of guessing.
- Conversations are multi-turn: follow-up questions understand the context.

Ask a question to get started.
